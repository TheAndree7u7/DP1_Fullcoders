ALGORITMO PlanificarLogisticaSimulatedAnnealing

ENTRADAS:
  - Lista de camiones disponibles
  - Lista de pedidos
  - Bloqueos de calles (no se aplican aún)
  - Camiones en mantenimiento

SALIDAS:
  - Asignación óptima de pedidos a camiones
  - Costo total de transporte
  - Lista de pedidos no entregados

INICIAR variables:
  temperatura ← 1000
  factorEnfriamiento ← 0.95
  iteracionesMaximas ← 1000

CARGAR:
  camiones ← obtenerCamiones()
  pedidos ← leerPedidosDesdeArchivo()

GENERAR solución inicial:
  solucionActual ← asignarPedidosAleatoriamente(camiones, pedidos)
  costoActual ← calcularCosto(solucionActual)
  mejorSolucion ← solucionActual
  mejorCosto ← costoActual

REPETIR iteracionesMaximas VECES:
  nuevaSolucion ← intercambiarUnPedido(solucionActual)
  nuevoCosto ← calcularCosto(nuevaSolucion)

  SI nuevoCosto mejora costoActual O se acepta por probabilidad THEN
    solucionActual ← nuevaSolucion
    costoActual ← nuevoCosto
    SI nuevoCosto < mejorCosto THEN
      mejorSolucion ← nuevaSolucion
      mejorCosto ← nuevoCosto
  FIN SI

  temperatura ← temperatura * factorEnfriamiento
FIN REPETIR

IMPRIMIR mejorSolucion y mejorCosto

IDENTIFICAR pedidos no entregados:
  entregados ← pedidos asignados en mejorSolucion
  noEntregados ← pedidos - entregados
  IMPRIMIR noEntregados

FIN ALGORITMO
